# Analyzing-a-Password-List-with-Python
In this assignment, you will write a Python program that reads a file containing usernames and passwords, then analyzes the passwords to find useful information.
Steps
1. Read the file into Python and store the passwords in a list.
2. Find the most frequently occurring password(s) (case sensitive). Count how many times each password appears and print the most common passwords(s) along with their count. "Password123" and "password123" should be counted as different passwords.
3. Find the most frequently occurring password(s) (case-insensitive). Count how many times each password appears and print the most common passwords(s) along with their count. For example, "Password123", "password123", and "PASSWORD123" should all be counted as the same password.
4. Find the shortest password, print it and include its length.
5. Find the longest password, print it and include its length.
5. Find the most frequently occurring letter in all passwords. What letter is it and how many times does it occur? A letter can occur more than once in a password. (Case senstive)
6. Find the least frequently occurring letter in all passwords. (Case in-sensitive) and print it and it's frequency.
7. Print all passwords that contain at least one uppercase character.
8. Print all passwords that are only lowercase, and do not include numbers.
9. Print a summary of your findings. Include: total number of passwords analyzed, most common password (case-insensitive) and it's count, longest password and it's length, number of passwords that contain at least one uppercase, and number of passwords that are only lowercase.
